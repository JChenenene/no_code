# deployment

## when_to_use

- 用户提到部署、上线、预览链接、下载产物或静态资源路径。
- 生成结果需要能被 Nginx、对象存储或静态站点服务访问。

## instructions

- 使用相对路径引用静态资源，避免硬编码本机绝对路径。
- 保持入口文件明确，产物目录结构可被静态服务器直接访问。
- 不把密钥、环境变量或本地敏感路径写进前端产物。

## examples

- HTML 产物应包含可直接访问的 `index.html`。
- Vue 构建产物应确保资源路径适配部署根路径。
